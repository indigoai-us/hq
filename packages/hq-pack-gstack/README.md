# @indigoai-us/hq-pack-gstack

HQ content pack: the gstack sprint team — 26 `g-*` skills plus the Codex bridge script.

## Install

```bash
hq install @indigoai-us/hq-pack-gstack
```

## Contents

- `workers/gstack-team/` — 26 `g-*` sprint skills (grooming, planning, standup, retro, etc.)
- `scripts/gstack-bridge.sh` — Codex skill bridge for gstack workers

## Requires

- `hq-core >= 12.0.0`
