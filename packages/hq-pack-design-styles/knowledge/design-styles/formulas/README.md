# Design Formulas

Repeatable recipes for specific design types. Each formula defines anatomy, section breakdown, style pairings, typography recipe, spacing, OSS stack, and a checklist.

## Categories

| Category | Formulas | Path |
|----------|----------|------|
| [Web](web/) | Landing page, SaaS marketing, Portfolio, Local business | `formulas/web/` |
| [App](app/) | Dashboard, Mobile app | `formulas/app/` |
| [Print](print/) | Business card, One-pager | `formulas/print/` |
| [Slides](slides/) | Pitch deck, Keynote | `formulas/slides/` |
| [Social](social/) | OG image, LinkedIn post, Instagram carousel | `formulas/social/` |

## How to Use

1. Pick the formula matching your design type
2. Pick a style from `styles/` for the visual language
3. Apply the formula's structure with the style's aesthetics
4. Reference `resources/` for OSS tool picks
