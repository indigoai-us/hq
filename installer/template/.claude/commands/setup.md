# Setup Wizard

Interactive setup for my-hq.

## Steps

1. Read the current agents.md file
2. Ask the user for their name, role, and goals
3. Update agents.md with their profile
4. Create any initial workers or projects they need
5. Explain next steps

## Prompt

Help me set up my-hq by asking about my profile and preferences. Update agents.md with my information and guide me through the initial configuration.
