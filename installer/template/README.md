# my-hq

Your personal AI operating system.

## Quick Start

1. Open a terminal in this directory
2. Run `claude`
3. Type `/setup` to configure your profile

## Documentation

See [USER-GUIDE.md](USER-GUIDE.md) for full documentation.

## Structure

- `.claude/` - Claude configuration and commands
- `agents.md` - Your personal profile
- `workers/` - AI worker definitions
- `projects/` - Project plans and PRDs
- `workspace/` - Session state and checkpoints
- `knowledge/` - Domain knowledge files
- `social-content/` - Content drafts

## Getting Help

Run `/help` in Claude to see available commands.
