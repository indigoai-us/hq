🔐 Authenticate Claude CLI

Claude CLI needs to be authenticated before use.

Open Terminal and run:

claude setup-token

This will open your browser to sign in to claude.ai

Getting Started

1
Open Terminal
Find it in Applications → Utilities → Terminal
2
Navigate to HQ
cd ~/hq
3
Start Claude
claude
4
Run the setup wizard
/setup
First time? When you run claude for the first time, you may be prompted to authenticate. Just follow the browser prompts to sign in with your Anthropic account.
Tip: If you see a "command not found" error for claude, try opening a new Terminal window or run source ~/.zshrc to reload your PATH.
