What This Installer Does

The HQ installer will:

1. Install HQ files to your home directory (~/hq)
2. Check for Node.js 18+ and install it if needed
3. Install the Claude CLI via npm (if not present)
4. Configure your shell PATH for easy access

After Installation

Once installation is complete:

  1. Open Terminal
  2. Navigate to HQ: cd ~/hq
  3. Start Claude: claude
  4. Run the setup wizard: /setup
Note: You'll need to authenticate with Claude on first run. The installer will attempt to open the authentication page automatically, but you may need to complete the process manually.

Uninstalling

To completely remove HQ:

  1. Delete the HQ folder: rm -rf ~/hq
  2. Optionally remove Claude CLI: npm uninstall -g @anthropic-ai/claude-code
  3. Remove PATH entries from ~/.zshrc or ~/.bash_profile

Getting Help

For help and support:

  • Run /help in Claude to see available commands
  • Read the USER-GUIDE.md in your HQ folder
  • Visit the GitHub repository for issues and discussions
