# Press Kit Checklist

## Required Assets

- [ ] **Press release** — AP style, under 600 words
- [ ] **Fact sheet** — Company/product key facts (1 page)
- [ ] **Executive bios** — 150-word bios for quoted executives
- [ ] **High-res logos** — PNG + SVG, light and dark versions
- [ ] **Product screenshots** — 3-5 key screens, high-res
- [ ] **Headshots** — Executive headshots, professional quality
- [ ] **Boilerplate** — Standard company description paragraph

## Optional Assets

- [ ] **FAQ** — 5-10 anticipated journalist questions with answers
- [ ] **Customer quotes** — 2-3 customer testimonials (with permission)
- [ ] **Data/research** — Proprietary data, survey results, benchmarks
- [ ] **Video** — Product demo or founder intro (under 90 seconds)
- [ ] **Infographic** — Visual summary of key data points
- [ ] **Comparison chart** — vs. competitors (for background, not publication)

## Distribution
- Host on company website at `/press` or `/media`
- Share via Google Drive link (not email attachments)
- Include link in pitch emails
- Update after each major announcement
