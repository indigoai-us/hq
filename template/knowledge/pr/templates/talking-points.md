# Talking Points Template

## Structure

```
# Talking Points: {{topic}}
**Prepared for:** {{spokesperson}}
**Date:** {{date}}
**Context:** {{interview/event/panel}}

## Key Messages (3 max)

### Message 1: {{headline}}
- **Statement:** {{1-2 sentence core message}}
- **Proof point:** {{data, example, or customer story}}
- **Bridge phrase:** {{transition back to this message}}

### Message 2: {{headline}}
- **Statement:** {{1-2 sentence core message}}
- **Proof point:** {{data, example, or customer story}}
- **Bridge phrase:** {{transition back to this message}}

### Message 3: {{headline}}
- **Statement:** {{1-2 sentence core message}}
- **Proof point:** {{data, example, or customer story}}
- **Bridge phrase:** {{transition back to this message}}

## Anticipated Questions

### Q: {{likely question}}
**A:** {{recommended answer}}

### Q: {{challenging question}}
**A:** {{recommended answer with bridge}}

### Q: {{off-topic question}}
**A:** {{bridge back to key messages}}

## Topics to Avoid
- {{sensitive topic and why}}

## Bridge Phrases
- "What's really interesting is..."
- "The bigger picture here is..."
- "What our customers tell us is..."
- "I think the real question is..."
```

## Rules
- Max 3 key messages (audience can't remember more)
- Each message must have a concrete proof point
- Anticipate at least 5 questions (2 friendly, 2 challenging, 1 off-topic)
- Bridge phrases help return to key messages when conversation drifts
- Keep answers under 30 seconds when spoken aloud
