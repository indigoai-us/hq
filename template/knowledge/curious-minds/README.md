# The Curious Mind's Handbook

A knowledge base extracted from "The Curious Mind's Handbook" -- a collection of 108 psychological, philosophical, and social concepts that give vocabulary to the fascinating phenomena surrounding us every day.

## What's Inside

Each concept includes:

- **What It Is** -- Clear definition of the concept
- **Origin** -- Where the idea came from and who developed it
- **Applications** -- Practical ways to use the concept in life, work, and relationships

## Structure

- `INDEX.md` -- Full table of all concepts with one-line descriptions
- `concepts/` -- One markdown file per concept, kebab-case naming

## Concept Count

**108 concepts** covering cognitive biases, logical fallacies, philosophical paradoxes, psychological phenomena, economic principles, linguistic curiosities, and social dynamics.

## Topics Span

- Cognitive biases (Dunning-Kruger, Survivorship Bias, Barnum Effect)
- Philosophical concepts (Allegory of the Cave, Ship of Theseus, Solipsism)
- Psychology (Learned Helplessness, Cognitive Dissonance, Placebo Effect)
- Economics (Pareto Principle, Veblen Goods, Nash Equilibrium)
- Social phenomena (Gaslighting, Emotional Contagion, Context Collapse)
- Cultural concepts (Ikigai, Wu Wei, Kintsugi, Schadenfreude)
- Technology and media (Algorithmic Bias, Hyperreality, Supernormal Stimuli)
- Logic and reasoning (Occam's Razor, Post Hoc Fallacy, Simpson's Paradox)
