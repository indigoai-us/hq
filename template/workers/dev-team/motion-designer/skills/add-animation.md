# add-animation

Add animation to a component.

## Arguments

`$ARGUMENTS` = `--component <name>` (required)

Optional:
- `--repo <path>` - Target repository
- `--type <entrance|exit|hover|loop>` - Animation type
- `--library <framer|gsap|css>` - Animation library

## Process

1. Analyze component structure
2. Design animation approach
3. Implement animation
4. Test performance
5. Present for approval

## Output

- Updated component with animation
- Animation configuration file (if needed)
