# learn

Capture learnings from task execution and inject them into relevant files.

## Usage

```
/learn
```

## Process

1. Analyze the current task
2. Extract learnings
3. Classify by scope
4. Inject into appropriate files

## Rules

- Always check for duplicates before adding a new rule
- Tier 1 rules go to CLAUDE.md Learned Rules section
- Never exceed the max rule count (20) in CLAUDE.md
