## Get started in 60 seconds

**Admin** — create a team:

```bash
npx create-hq
# Choose "Create an HQ Teams account"
# Pick your GitHub org, name your team
# Invite members when prompted
```

**Member** — accept an invite:

```bash
npx create-hq --invite hq_<your-token>
# Sign in with GitHub
# Team content cloned automatically
```

## How it works

| Step | Admin | Member |
|------|-------|--------|
| **1. Create** | Runs `npx create-hq`, picks GitHub org, names team | -- |
| **2. Invite** | Generates invite token + sends GitHub org invite | Receives email + token |
| **3. Join** | -- | Runs `npx create-hq --invite hq_<token>` |
| **4. Collaborate** | Pushes content to shared repo | Pulls latest with sync |
| **5. Grow** | Invites more members | Invites more members (if admin) |

## Requirements

- **Node.js** 18+ (LTS recommended)
- **Git** 2.25+ (for sparse checkout support)
- **GitHub account** (free tier is fine)
- **GitHub organization** (admin creates one — free for public repos, Teams plan for private)
