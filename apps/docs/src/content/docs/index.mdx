## What's in the box

| Component | Description |
|-----------|-------------|
| **39+ Commands** | Session management, workers, projects, content, system, and more |
| **28+ AI Workers** | Dev team, content team, social team, PR team, QA, security, and more |
| **44+ Skills** | Specialized capabilities across the full skill tree |
| **132+ Policies** | Learned rules, safety guardrails, and workflow enforcement |
| **20+ Knowledge Bases** | Ralph methodology, security, design styles, testing, worker framework |
| **3 Starter Projects** | Personal assistant, social media, code worker |
| **Module System** | Keep your HQ updated with upstream changes |

## HQ vs HQ Cloud

HQ is **free and open-source** (MIT). HQ Cloud adds managed infrastructure for teams and mobile access.

| Feature | HQ (Free) | HQ Cloud (Paid) |
|---------|:---------:|:----------------:|
| All Claude Code commands | Yes | Yes |
| AI workers & skills | Yes | Yes |
| Knowledge bases & policies | Yes | Yes |
| Session management & checkpoints | Yes | Yes |
| qmd semantic search | Yes | Yes |
| `npx create-hq` installer | Yes | Yes |
| Module system & updates | Yes | Yes |
| Cloud sync (S3) | -- | Yes |
| Mobile PWA dashboard | -- | Yes |
| Managed hosting | -- | Yes |
| Team collaboration | -- | Roadmap |
| Priority support | -- | Yes |
