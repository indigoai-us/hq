# Example Skill

A sample skill showing the basic structure. Skills are markdown files that define what a worker can do.

## Context to Load

1. Read the task from the project PRD
2. Load any relevant knowledge files

## Steps

1. Analyze the task requirements
2. Plan the implementation approach
3. Execute the work
4. Run verification checks
5. Write a checkpoint with `/checkpoint`

## Rules

- Always verify before marking complete
- Ask for clarification rather than guessing
- Write a checkpoint after each completed task

## Output

Describe what this skill produces (e.g., code changes, reports, content drafts).
