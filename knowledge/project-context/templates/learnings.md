## How to Add a Learning

1. Identify the appropriate section (Patterns, Gotchas, Tips, Open Questions)
2. Use the format: `### [YYYY-MM-DD] [worker-name] [task-id]: {Title}`
3. Include the category tag for filtering
4. Add an entry to the index at the top
5. Be specific - include context, examples, and actionable advice

**Categories for filtering:**
- `pattern` - Approaches that work well
- `gotcha` - Things to watch out for
- `tip` - Quick helpful tricks
- `question` - Unresolved issues
- `performance` - Performance-related insights
- `testing` - Testing-related learnings
- `integration` - Integration-related learnings
- `security` - Security-related learnings
