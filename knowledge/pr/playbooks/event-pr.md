# Event PR Playbook

## Event Types
1. **Conference speaking** — Founder speaking at industry events
2. **Product demo events** — Webinars, live demos, launch events
3. **Industry events** — Attending as exhibitor or participant
4. **Hosted events** — Company-organized meetups, roundtables

## Timeline

| Day | Activity |
|-----|----------|
| D-21 | Draft media alert |
| D-14 | Build event-specific media list (local + trade press) |
| D-14 | Send media alert to press |
| D-7 | Pitch 1:1 briefings/interviews at event |
| D-3 | Prep talking points for spokesperson |
| D-1 | Send reminder to confirmed press |
| D-0 | Live coverage: social posts, photo documentation |
| D+1 | Send follow-up to press who attended |
| D+3 | Send recap + key announcements to those who didn't attend |
| D+7 | Coverage report |

## For Conference Speaking
- Pitch journalists for pre-event briefings
- Draft social thread summarizing talk key points
- Record or get recording of talk for future use
- Post talk highlights as social content for 1-2 weeks after

## Deliverables
1. Media alert (send via `templates/media-alert.md`)
2. Talking points for spokesperson
3. Social content plan (pre, during, post event)
4. Coverage report
